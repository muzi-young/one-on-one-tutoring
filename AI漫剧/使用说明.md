## 🎯 不同场景的使用建议

### 场景 1：客户初次拜访 / 第一轮汇报
**目标**：让客户感受到我们的专业度，建立信任，引出需求。

推荐顺序：
1. **第 1 步**：用 `AI漫剧行业研究报告.html` 切入，30 分钟讲清赛道、市场、商业模式（让客户认同方向）。
2. **第 2 步**：演示 `AI漫剧智造平台-交互原型.html` 中的 1-2 个核心页面，让客户「看到产品」。
3. **第 3 步**：留下 `行业研究报告.html` 给客户研究，约下一轮深聊。

### 场景 2：客户第二轮 / 决策层汇报
**目标**：拿下项目意向、推进合同。

推荐顺序：
1. **第 1 步**：直接用 `AI漫剧智造平台-项目报价方案.html` 完整过 6 大模块（约 60-90 分钟）。
2. **第 2 步**：穿插原型 Demo，让客户感受可落地性（重点演示 C6 图生视频工坊 / C5 分镜板 / M2 内容审核 三个差异化页面）。
3. **第 3 步**：现场报价（旗舰版 ¥48.78 万），给客户三档对比（标准 / 旗舰 / 私有化）便于客户选择。

### 场景 3：自身团队启动会 / 立项评审
**目标**：让产研团队理解项目背景与边界。

推荐顺序：
1. 先发 `AI漫剧行业资料整理.md` 给所有人提前阅读。
2. 启动会现场过 `项目报价方案.html` 中的「需求分析 → 功能清单 → 技术方案」三个模块。
3. 演示 `交互原型.html` 帮助产品/UI/前后端对齐。

## 🛠️ 文件打开技巧

### HTML 文件如何看起来更好？
- **推荐使用 Chrome / Edge / Safari** 最新版本打开。
- 在 PC 上打开效果最佳（报价方案有侧边目录、原型有大屏幕展示）。
- 报价方案 / 研究报告支持 `Ctrl+P` 一键打印为 PDF（已优化打印样式）。
- 交互原型在大屏幕上一次能看到 3-4 部手机模型，演示效果最佳。

### Markdown 文件如何编辑？
- **推荐 Typora**（所见即所得）或 **VSCode** + Markdown 插件。
- 也可以直接复制粘贴到飞书文档 / 钉钉文档 / 腾讯文档，保留完整格式。
- 如果需要导出为 Word，Typora 的「导出 → docx」即可。

## 💡 客户最常问的问题（FAQ 应答模板）

### Q1：你们这个平台和市面上的 AImate / 巨日禄 / 扑克 AI 有什么区别？
**答**：我们的差异化在于五个点（详见报价方案 0.2 节）：
1. **多模型聚合 + 智能调度**：5 模型动态选择最优，调用成本降 30-40%；
2. **角色一致性引擎**：自研 LoRA 引擎，跨集人物零漂移（行业最大痛点）；
3. **多人在线协同**：类 Figma 体验，工作室效率 ↑3 倍；
4. **创作发行一体**：一键分发抖快/视频号/自有 App；
5. **私有化部署**：满足国企 / 影视集团合规要求。

### Q2：48 万的报价是怎么核算出来的？
**答**：按行业标准计费规则严格测算（详见报价方案第 6 章）：
- Java 后端 142 人天 × 1800 元 = 25.56 万；
- uni-app + Vue 前端 66 人天 × 1200 元 = 7.92 万；
- UI 设计 32 人天 × 1000 元 = 3.2 万；
- 项目管理（10%）+ 风险缓冲（10%）= 7.34 万；
- 含税总价 ≈ 48.78 万元。

### Q3：私有化部署的成本是多少？
**答**：私有化版本 88 万起，含旗舰版完整功能 + 私有化部署 + LoRA 自训练 + 30 天定制开发。具体看客户业务定制需求量。

### Q4：你们能保证视频生成质量吗？
**答**：质量取决于模型选择 + 提示词 + 角色一致性引擎。我们的多模型聚合策略可以让单镜头自动选择最优模型，配合自研 LoRA 实现 95%+ 角色一致性，整体可达「商用可发布」水平（参考案例：单部成本 5 万 / 平均分账 50-150 万，ROI 10-30 倍）。

### Q5：合规风险怎么控制？
**答**：四道闸门（详见报价方案技术难点 D6）：
1. AI 多模态审核（画面+文本+音频）；
2. 关键词敏感词库 + 模板库；
3. 人工终审 + 申诉机制；
4. 上线前自检报告。  
所有 AI 模型优先采用国内合规供应商（可灵、即梦、海螺、豆包），数据可私有化部署。

### Q6：项目周期为什么需要 6 个月？
**答**：120 个工作日已经是含三端 + 协同 + 多模型聚合的高效进度（详见排期 §07）。常规对比：
- 同等复杂度的电商平台 ≥ 8 个月；
- 同等复杂度的内容平台 ≥ 7 个月；  
我们已经通过通用基础组件 + AI 调度框架进行了大量复用。
