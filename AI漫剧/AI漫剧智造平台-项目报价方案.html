00产品概述

一句话讲清楚我们要建一个什么样的平台。
AI 漫剧智造平台是面向 AI 漫剧创作者、工作室、MCN 与影视集团的一站式 SaaS 平台， 通过聚合可灵 / 即梦 / 海螺 / Vidu / 通义万相等国内主流 AI 视频模型与豆包 / 讯飞 / CosyVoice等语音模型， 打通"剧本生成 → 角色设计 → 分镜规划 → 图生视频 → 配音口型 → 自动剪辑 → 多端分发"全流程，让 AI 漫剧的工业化生产门槛从"专业团队"降至"个人/小工作室"。

01需求分析报告

从客户业务出发，把"为什么要做这件事"说透。

1.1 需求背景

2025 年中国微短剧市场规模突破 680 亿元，AI 短剧/漫剧细分超 100 亿元，2027 年预计达 380 亿元，年复合增长率超 100%。当前痛点集中在：

  • 真人短剧成本高（5-15 万/集）、周期长（7-15 天/集），爆款命中率持续走低；
  • 市面上的 AI 漫剧 SaaS 工具普遍存在角色一致性差、流程割裂、协同弱、合规风险四大痛点；
  • 红果 / 星芽等头部分发平台对 AI 漫剧明确扶持，但优质创作者供给严重不足；
  • 影视集团 / MCN 急需"工业化产能 + 数据资产私有"的内部生产平台。

1.2 核心诉求（显性 + 隐性）

类别诉求点说明
显性诉求降本增效单集成本降至 1000 元以内，周期压缩到 1-3 天
多模型聚合统一接口调度可灵/即梦/海螺等 5+ 模型，动态选择最优
三端一体创作者端 + 观众端 + 管理后台一体化交付

D1 多 AI 模型聚合调度 5+ 模型 API 协议不统一、计费不同、稳定性差异大、单镜头需要选最优 ① 自研统一 AI 网关（标准化 Prompt 协议）；② 模型能力画像 + 智能路由（按场景/角色/动作选优）；③ 失败重试 + 降级；④ 实时算力监控与配额。
D2 角色一致性引擎 跨集 / 跨分镜五官、服装、风格漂移；行业最大痛点 ① 自研 LoRA 训练流水线 + 角色资产库；② 即梦/可灵的「角色参考」API 调度；③ 多图垫图（i2i）+ ID 锁定；④ 重生时自动比对相似度。
D3 大文件视频处理与 CDN 每集 30-60 个视频片段、单段 50-200MB，并发上传/转码/分发压力大 ① OSS 分片上传 + 断点续传；② 视频转码集群（FFmpeg + GPU）；③ 全国 CDN + 智能调度；④ 自适应清晰度（HLS/DASH）。
D4 剧本/分镜多人在线协同 多人同时编辑、版本冲突、操作回放 ① CRDT/OT 算法做协同编辑；② 操作日志事件溯源；③ WebSocket 实时同步；④ 段落锁机制；⑤ 历史版本回溯。
D5 中文口型对齐 中文音节与口型映射难、机械感重、多角色场景失控 ① 优先调度即梦/可灵/海螺 02 原生中文对口型；② 自部署 Sadtalker/Wav2Lip 兜底；③ 关键帧人工微调入口；④ 多人对话拆为单镜头反打。
D6 内容审核合规 涉黄/涉暴/涉政/广告法风险，下架可致全平台封禁 ① AI 多模态审核（画面+文本+音频）；② 关键词库 + 敏感模板库；③ 人工终审 + 申诉机制；④ 操作留痕；⑤ 上线前自检报告。
D7 高并发观众端播放 爆款剧上线峰值 10 万+ 在线播放，CDN 与播放器压力大 ① 多级缓存（CDN+边缘节点+客户端）；② 自适应清晰度；③ 用户分桶预热；④ 限流熔断；⑤ Redis 集群应对热数据。
D8 财务结算与多端分账 创作者分成 + 平台分账 + 工作室二级分成 + 税务发票 ① 分账规则引擎（阶梯/补贴/扣款）；② 分布式事务（Seata/TCC）；③ 月结对账系统；④ 自动开具发票（百望云/航信）；⑤ 提现审批流。
