Part 01

产品概述

古琴垂直领域智能体系统，串联学习内容运营、智能问答、哼唱识曲与推荐、数字人视频与直播，以及课程卡片与优惠转化。系统由用户端 APP、运营端、管理后台与 AI 服务能力共同构成，数据侧覆盖知识素材、曲库与样本、问答与识别记录、直播互动与转化统计。

建设目标

  • 通过古琴知识库问答承接学习咨询、课程咨询与常见问题，降低人工答疑成本。
  • 打通问答、识曲、曲目详情、直播互动与课程推荐，形成内容到销售转化的路径。
  • 通过数字人视频生成能力支撑运营批量产出课程介绍、知识科普与活动推广素材。
  • 以哼唱识曲、相似曲目与课程推荐、曲库管理构建差异化 AI 体验与专业度。
  • 通过数字人直播、弹幕互动、课程卡片与优惠配置承接私域获客与直播卖课。
Part 02

需求分析

维度说明
需求背景古琴教培与内容运营需要在垂直场景内统一「答疑—内容—识曲—直播—卖课」链路；人工成本高，需 AI 与流程化后台支撑规模化运营。
核心诉求智能问答与语音交互、课程视频推荐与学习进度、哼唱识曲与曲目/课程推荐、数字人视频与直播、弹幕理解与即时应答、优惠与课程卡片转化、全链路后台与数据统计。
目标用户画像C 端：古琴学习者与潜在购课用户；B 端：运营人员（视频与直播策划、活动配置）；管理员：内容审核、曲库与授权、数字人资产与风控规则维护。
交付范围（三期 MVP）用户端 APP：一期问答与课程推荐；二期哼唱识曲与匹配推荐；三期直播入口、弹幕互动与课程转化。运营端：一期数字人视频生成；三期直播运营与转化配置。管理后台：一期知识库与课程、数字人素材；二期曲库与相似推荐、音频匹配管理；三期数字人资产、弹幕审核规则与直播统计。AI 服务：问答 RAG、音频匹配、数字人驱动、弹幕理解与异常降级。
关键假设现有 APP 可按约定技术栈接入或以内嵌页方式集成；大模型、语音识别、数字人与直播能力以国内合规第三方成熟服务集成为主；曲库与版权素材由甲方提供并完成授权背书。

非功能性需求

类别要求
性能问答与列表类接口响应满足常规移动端体验；生成类与直播类任务异步队列化，状态可查询；高峰期通过队列与限流保障可用性。
安全后台权限分级；操作日志与关键配置留痕；敏感词与弹幕风控可配置；用户内容与授权字段可追溯。
兼容用户端以甲方现有 APP 载体为准完成接入；管理后台与运营端以主流 PC 浏览器可用为目标。
合规云服务与 AI、直播、支付等三方选用国内合规供应商；内容由甲方负责版权与授权合规，系统侧记录授权与备注字段。

Part 04

技术选型

后端优先 PHP；用户端与后台前端优先 Vue；APP 侧以接入既有容器为准（含 H5 内嵌）。AI 与音视频能力以成熟国内合规云服务及供应商 API 集成为主。

层级选型理由
后端PHP业务接口、权限、任务队列、统计报表与三方回调聚合；交付节奏与运维成本低，适配中小型垂直业务快速迭代。
前端（移动端）Vue组件化交付 H5/内嵌模块或与原生壳通信页面；与 PC 后台技术栈统一，降低联调与人力切换成本。
管理后台 / 运营端（PC Web）Vue表格、表单、审核流与配置类页面密集场景下维护性好；与移动端共用设计语言与组件沉淀。
UI 设计全套视觉稿覆盖 APP 核心路径与后台关键页面，输出交互规范与标注，保障多端一致性。

三方服务集成（本项目涉及类别）

服务类别推荐厂商备选厂商推荐理由
云服务器 / 弹性计算腾讯云 CVM阿里云 ECS、华为云 ECS与对象存储、CDN、直播生态联动成本低；控制台与备案链路成熟。

Part 05

技术难点与解决方案

难点解决方案要点
问答幻觉与无来源答复风险检索增强生成 + 来源段落绑定展示；低置信度触发兜底话术或转人工入口；后台问答审核与知识标签约束。
哼唱识曲准确率波动曲库样本标准化与分批导入；指纹/特征匹配规则可配置；结果以「候选 + 相似度 + 不确定提示」为验收口径，试运行数据驱动阈值调优。
数字人直播稳定性与时延脚本化播报主路径 + 异步弹幕理解插入；异常降级到预设话术 / 人工接管 / 回放垫片；服务商 SLA 与重连策略前置验证。
直播弹幕风控与体验平衡敏感词与频率限制可配置；人工审核开关与黑名单；违规分类联动屏蔽策略。
与既有 APP 账号 / 支付 / 课程体系对接启动阶段冻结接口契约与联调环境；Mock 服务并行开发；字段映射表与版本兼容策略文档化。
版权与授权合规留痕曲目与素材后台强制维护授权字段；未授权状态限制前台曝光或加水印提示（规则由甲方确认）。
Part 06

技术实施方案

整体采用前后端分离：移动端与 PC 端通过 HTTPS 调用统一业务 API；异步任务（生成、识别、直播辅助）经消息队列或任务表驱动工作进程；AI 与直播相关能力通过适配层对接第三方。数据层采用关系型数据库承载业务实体，对象存储承载音视频与素材文件。

表现层
现有 APP 容器 + H5/Vue 页面 运营端 Vue（PC） 管理后台 Vue（PC）
服务层
PHP 业务 API（REST） 权限与审计中间件 任务队列（生成 / 识别） AI 适配层（RAG / ASR / 数字人 / 弹幕 NLP） 第三方 Webhook 接入
数据层
关系型数据库（业务与配置） 对象存储 COS/OSS（音视频与素材） 检索索引（知识库 / 曲库特征） 日志与运营报表明细表
交付层
部署文档与环境变量说明 接口说明与回调约定 验收用例与培训材料 上线运维交接清单

方案边界说明

  • 不包含自研大模型训练；采用模型 API + 知识库检索增强。
  • 不包含从零自研数字人底层引擎；采用第三方数字人 SaaS/SDK。
  • 不包含高精度音乐识别算法从零研发；MVP 采用特征库匹配与可配置规则，可选叠加第三方识别。
  • 不包含甲方历史 APP 大规模架构重构；超出范围的改造单独评估。
  • 不包含云资源费、模型按量费、直播流量费、短信推送费及版权采购费。
