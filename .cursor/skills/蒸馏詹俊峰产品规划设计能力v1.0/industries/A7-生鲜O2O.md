# A7 - 生鲜O2O

## L1 领域核心模型

### 核心角色
- **消费者**：购买生鲜食材的个人或家庭用户。
- **前置仓/门店**：负责生鲜存储、分拣和最后配送的节点。
- **平台运营方**：管理供应链、库存和配送。
- **产地供应商**：提供生鲜货源的农场、果园、渔场等。

### 核心资源
- **冷链物流**：从产地到消费者手中的全程温控运输。
- **库存管理**：生鲜效期短，库存周转决定损耗率。
- **品控标准**：生鲜非标品，需要统一的品质分级标准。

### 核心诉求
- **消费者**：买到新鲜安全的食材，送达时商品完好，不缺斤少两。
- **平台**：降低损耗率，提高周转速度，控制冷链成本。
- **供应商**：稳定的销售渠道，不被恶性压价。

### 关键矛盾
- **新鲜度与时效**：消费者要最新鲜的，但配送时间越长新鲜度越低。
- **库存与损耗**：备货少了缺货，备货多了过期，预测难度大。
- **非标品与标准化**：每颗苹果大小不一，消费者和平台对品质的认知可能不同。
- **价格波动**：生鲜受天气和季节影响大，价格波动频繁。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：前置仓+即时配送
- **场景描述**：城市密布小型前置仓，用户下单后30分钟配送到家。
- **关键特性**：时效快、履约成本高、SKU 受限于仓容。

### 形态二：店仓一体（新零售）
- **场景描述**：既有线下门店可供消费，同时门店也是线上订单的发货仓。
- **关键特性**：线上线下一体化、库存共享、用户体验多元。

### 形态三：社区团购配送
- **场景描述**：用户通过团长下单，次日达团长自取点。
- **关键特性**：履约成本低、时效不如即时配、适合计划性采购。

### 形态四：产地直发
- **场景描述**：用户下单后，从产地直接快递发货。
- **关键特性**：适合耐储运品类（如水果），中间成本低，但配送时间 2-4 天。
