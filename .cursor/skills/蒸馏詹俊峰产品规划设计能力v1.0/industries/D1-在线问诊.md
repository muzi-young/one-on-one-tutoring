# D1 - 在线问诊

## L1 领域核心模型

### 核心角色
- **患者/咨询者**：有健康问题需要咨询医生的个人。
- **医生**：提供在线诊疗意见的执业医师。
- **平台运营方**：匹配医患、提供问诊工具、管理服务质量。
- **监管机构**：卫健委等政府部门，监督互联网诊疗合规。

### 核心资源
- **医生供给**：足够数量和质量的在职医生，是平台的核心竞争力。
- **问诊流程**：图文、电话、视频等问诊方式的标准化流程。
- **电子病历**：患者健康档案和问诊记录的沉淀。

### 核心诉求
- **患者**：方便快速地获得专业医疗意见，小病不用去医院排队。
- **医生**：利用碎片时间获得额外收入，提高个人品牌知名度。
- **平台**：撮合医患匹配，控制医疗风险，通过问诊费抽佣和增值服务变现。

### 关键矛盾
- **问诊效率与诊疗质量**：快速接单提升效率，但可能因为信息不足导致误诊。
- **轻问诊与严肃医疗**：在线只能解决轻症和咨询，重症必须引导至线下，分诊精准度是关键。
- **处方合规与用户便利**：开药需求强，但处方合规要求严格。
- **医生时间与平台控制**：医生是稀缺资源，平台对头部医生的议价能力弱。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：图文/电话/视频混合问诊
- **场景描述**：支持多种问诊方式，患者根据紧急程度和需求选择。
- **关键特性**：图文成本低覆盖面广，视频适合需要视诊的场景。

### 形态二：AI 预问诊+医生复核
- **场景描述**：AI 先收集症状信息并给出初步判断，医生在此基础上复核和追问。
- **关键特性**：提升医生效率，缩小误诊风险，适合常见病和慢病咨询。

### 形态三：专科垂直问诊
- **场景描述**：聚焦某一专科（如皮肤、心理、儿科），医生专业度更高。
- **关键特性**：用户精准匹配，服务质量更可控，天花板取决于专科人群规模。

### 形态四：家庭医生/私人医生模式
- **场景描述**：用户购买长期服务，固定医生团队持续关注全家健康。
- **关键特性**：用户粘性极高，医患信任深，需要医生的持续时间投入。

### 形态五：医联体/远程会诊模式
- **场景描述**：基层医生在线向上级医院专家发起会诊请求。
- **关键特性**：B2B 属性强，提升基层医疗水平，与医院深度合作。
