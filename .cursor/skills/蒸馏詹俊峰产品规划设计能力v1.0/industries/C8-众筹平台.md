# C8 - 众筹平台

## L1 领域核心模型

### 核心角色
- **项目发起人**：有创意或项目需要资金，在平台上发起众筹的个人或团队。
- **支持者/投资者**：浏览项目、出资支持、获得回报的用户。
- **平台运营方**：审核项目真实性，管理资金流转，提供展示和沟通工具。

### 核心资源
- **优质项目**：有创意、有吸引力的项目是平台的核心内容。
- **社区信任**：支持者相信项目发起人会按承诺交付回报。
- **资金托管能力**：合理的资金发放机制（一次性/里程碑式）保障支持者权益。

### 核心诉求
- **发起人**：获得启动资金，验证市场对产品的需求，积累早期用户。
- **支持者**：支持有意义的项目，获得预期回报，产品或体验符合预期。
- **平台**：促成更多成功的众筹项目，通过交易抽佣或增值服务变现。

### 关键矛盾
- **项目质量与成功率**：降低门槛让更多新奇想法进入，但跳票过多会损害平台声誉。
- **创意保护与公开展示**：项目需要公开展示以吸引资金，但完全没有专利保护的早期创意容易被抄袭。
- **奖励驱动的冲动支持 vs 冷静评估**：更多支持来自一时激动和感性冲动，这增加了项目逾期和烂尾的风险。
- **全球化与本地合规**：跨境众筹能扩大项目和支持者范围，但各国证券法和消费者保护法差异大。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：回报型众筹
- **场景描述**：支持者出资后获得实物产品或服务作为回报（非股权、非利息）。
- **关键特性**：模式最简单，合规门槛较低，类似预售。

### 形态二：股权型众筹
- **场景描述**：支持者出资后获得项目公司的股权份额。
- **关键特性**：合规门槛极高（受证券法监管），支持者需合格投资者认证。

### 形态三：捐赠型众筹
- **场景描述**：支持者捐款支持公益或个人求助项目，不期待物质回报。
- **关键特性**：情感驱动，需要强社会信任和透明的资金流向公示。

### 形态四：创意订阅/会员制众筹
- **场景描述**：支持者按月付费，定期获得创作者的内容或福利。
- **关键特性**：持续性收入，创作者与粉丝的关系维护是核心。
