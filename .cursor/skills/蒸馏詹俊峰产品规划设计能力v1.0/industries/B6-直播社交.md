# B6 - 直播社交

## L1 领域核心模型

### 核心角色
- **主播**：通过实时视频与观众交流互动的人，提供表演、聊天或知识分享。
- **观众**：观看直播并参与弹幕、打赏互动的用户。
- **公会/MCN**：签约和管理主播的机构，负责培训和商业变现。
- **平台运营方**：提供直播技术、流量分配、礼物系统和活动运营。

### 核心资源
- **主播**：优质主播是直播平台的核心资产，头部主播有强大的流量号召力。
- **互动氛围**：弹幕、礼物、连麦等实时互动带来的群体参与感。
- **礼物体系**：虚拟礼物的设计和定价直接影响打赏收入。

### 核心诉求
- **主播**：获得人气和收入，建立自己的粉丝群体，持续成长。
- **观众**：获得娱乐消遣、情感陪伴、或知识学习，与主播建立连接。
- **平台**：最大化用户观看时长和打赏流水，维持主播生态的活力。

### 关键矛盾
- **头部主播与平台博弈**：头部主播自带流量，有较强议价能力，一旦跳槽或停播影响平台收入。
- **诱导打赏与理性消费**：直播间的气氛和主播话术可能诱导非理性打赏，舆论风险高。
- **监管与内容自由**：实时直播内容审核难度大，违规内容一旦播出监管处罚重。
- **内容同质化**：大量直播内容无创新，依赖颜值和套路，用户审美疲劳。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：泛娱乐直播
- **场景描述**：以才艺表演、闲聊互动、户外直播为主要内容。
- **关键特性**：主播颜值和表达能力是核心，打赏是主要变现，竞争激烈。

### 形态二：游戏直播
- **场景描述**：主播直播自己玩游戏的过程，观众通过弹幕和打赏互动。
- **关键特性**：游戏内容和主播水平是关键，赛事直播有强周期性流量。

### 形态三：垂直技能直播
- **场景描述**：主播教授某种技能（如唱歌、绘画、编程），观众付费学习。
- **关键特性**：内容有深度，用户付费意愿强，适合教育类变现。

### 形态四：1v1 社交直播
- **场景描述**：用户随机匹配陌生人进行一对一视频聊天。
- **关键特性**：社交属性强，用户愿意为匹配次数和隐私保护付费，内容审核压力大。
