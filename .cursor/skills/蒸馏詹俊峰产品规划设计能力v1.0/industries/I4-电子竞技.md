# I4 - 电子竞技

## L1 领域核心模型

### 核心角色
- **职业选手/战队**：经过专业训练，参加电竞赛事的顶尖玩家和战队组织。
- **赛事组织方**：主办、承办电竞比赛的联盟或赞助商。
- **观众/粉丝**：观看电竞赛事、关注选手动态的电竞爱好者。
- **平台/直播方**：线上直播赛事、提供数据分析和社区讨论。

### 核心资源
- **游戏版权**：举办某款游戏的电竞赛事需要游戏厂商授权。
- **选手与战队 IP**：明星选手和热门战队的号召力是票房和流量的保证。
- **赛事制作能力**：直播包装、数据分析、解说和场馆运营。

### 核心诉求
- **选手/战队**：赢得荣誉和高额奖金，获得赞助和粉丝支持。
- **赛事方**：打造有影响力的赛事品牌，获取门票、赞助和转播收入。
- **观众**：观看高水平比赛，支持喜欢的选手，参与社区讨论。
- **平台**：获得高并发直播流量，推动游戏本身的活跃和付费。

### 关键矛盾
- **游戏版权方与赛事独立性**：游戏公司掌握赛事生杀大权，赛事生态高度依附游戏。
- **竞技公平与商业化**：涉及假赛、博彩等问题，一但爆雷影响整个赛事公信力。
- **新游戏迭代与老游戏淘汰**：电竞主流项目迭代缓慢，新游戏很难挑战老牌热门项目。
- **选手青春期极短**：多数选手在25岁前退役，职业生涯短暂，后路保障需要体系支持。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：联赛制职业电竞模式
- **场景描述**：常驻席位制度，赛季常规赛加季后赛。
- **关键特性**：联盟化管理，收入与席位费、转播权、商业赞助密不可分。

### 形态二：杯赛/公开赛模式
- **场景描述**：各战队通过预选赛进入正赛，单淘汰制决定冠军。
- **关键特性**：爆冷和黑马多，观赏性强，周期较短。

### 形态三：全民赛/社区赛模式
- **场景描述**：向普通玩家开放报名，线上选拔，大型赛事总决赛同台。
- **关键特性**：玩家转化率高，对游戏本身活跃度促进作用显著。

### 形态四：电竞赛训 SaaS 模式
- **场景描述**：为战队提供训练赛数据分析、录像复盘、教练工具。
- **关键特性**：服务 B 端，与竞技水平提升挂钩，商业闭环清晰。
