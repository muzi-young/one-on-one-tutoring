# H4 - 数字阅读小说

## L1 领域核心模型

### 核心角色
- **读者**：通过手机或电子书阅读器阅读小说的用户。
- **作者/作家**：创作连载或完本小说的个人。
- **平台运营方**：提供阅读工具、内容分发和付费体系的技术方。

### 核心资源
- **内容版权**：签约作者生产的独家内容。
- **阅读体验**：字体、背景、翻页动效、听书功能的体验打磨。
- **读者数据**：阅读进度、偏好分类、付费意愿等行为数据。

### 核心诉求
- **读者**：找到喜欢看的小说，阅读体验舒适，价格合理。
- **作者**：获得稿费和读者认可，签约平台能提供稳定的写作保障。
- **平台**：做大付费阅读规模，通过付费分成和广告变现。

### 关键矛盾
- **免费与付费**：免费阅读吸引大量用户，但作者收入依赖广告分成不如付费稳定。
- **盗版与版权保护**：小说盗版成本低，正版平台内容很容易被爬取。
- **内容审核与创作空间**：严格审核可能限制作者发挥，太宽松则有内容风险。
- **推荐公平与新书曝光**：流量集中在头部作品，新人和小众作品难以出头。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：付费章节模式
- **场景描述**：前几章免费，后面章节按千字付费阅读。
- **关键特性**：作者收入与作品质量直接挂钩，激励创作好作品。

### 形态二：会员包月模式
- **场景描述**：用户支付月费，全场小说免费读。
- **关键特性**：阅读量大的用户划算，平台需控制作者分成成本。

### 形态三：免费+广告模式
- **场景描述**：读者免费阅读，通过插页广告和激励视频获取收入。
- **关键特性**：用户基数可以做到很大，适合下沉市场。

### 形态四：有声书/广播剧融合模式
- **场景描述**：同一部小说的文字和有声版本同步上线。
- **关键特性**：丰富消费场景（通勤听书），制作成本高于纯文字。
