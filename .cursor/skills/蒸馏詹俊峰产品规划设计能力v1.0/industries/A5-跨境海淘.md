# A5 - 跨境海淘

## L1 领域核心模型

### 核心角色
- **消费者**：购买海外商品的国内用户。
- **平台运营方**：对接海外供应商，管理跨境物流和清关流程。
- **海外商家/供应商**：提供海外商品货源的品牌或经销商。
- **海关/监管部门**：执行跨境贸易法规，征收关税，查验禁限物品。

### 核心资源
- **海外供应链**：稳定的海外直采或合作渠道。
- **清关能力**：高效合规的通关流程，直接影响用户收货时效。
- **正品信任**：消费者对海外商品真伪的高度关注。

### 核心诉求
- **消费者**：买到海外正品，价格合理，物流较快，不被假货坑害。
- **平台**：确保正品供应链，降低退货率，处理复杂的跨境退换货。
- **海外商家**：进入中国市场，控制跨境物流成本。

### 关键矛盾
- **速度与成本**：最快的物流方式（空运直邮）成本高，最便宜的方式（海运保税）等待长。
- **正品与假货**：跨境链路过长，假货容易混入，查验成本高。
- **合规与便利**：海关政策多变，严格合规增加流程复杂度，过于简化可能违规。
- **退换货困境**：跨境退换货的物流和关税成本极高，用户体验差。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：保税仓模式
- **场景描述**：商品先批量运至国内保税仓，用户下单后从保税仓发货。
- **关键特性**：物流时效接近国内快递，只能销售已备货的 SKU。

### 形态二：直邮模式
- **场景描述**：用户下单后，商品从海外仓库直接邮寄至用户手中。
- **关键特性**：SKU 丰富，长尾商品可售，物流时效长，运费高。

### 形态三：买手/代购模式
- **场景描述**：个人买手在海外采购，通过平台连接国内消费者。
- **关键特性**：非标品多，正品信任依赖买手个人信用，合规风险高。

### 形态四：分销代发模式
- **场景描述**：国内卖家通过 API 对接海外供应商，用户下单后由供应商直发。
- **关键特性**：轻资产、零库存，但对供应链掌控力弱。
