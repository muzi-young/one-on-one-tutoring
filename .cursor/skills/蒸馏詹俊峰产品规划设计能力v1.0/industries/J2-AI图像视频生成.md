# J2 - AI图像视频生成

## L1 领域核心模型

### 核心角色
- **创作者**：使用 AI 生成图片或视频的设计师、艺术家、营销人员、普通用户。
- **模型提供方**：训练和托管图像/视频生成模型的公司。
- **平台运营方**：提供提示词输入、生成队列、作品管理和社区分享。
- **版权/合规方**：关注 AI 生成内容的版权归属和深度伪造风险。

### 核心资源
- **生成模型**：基于扩散模型、GAN 等技术的文生图、图生视频模型。
- **算力资源**：GPU 推理能力，直接决定生成速度和并发能力。
- **用户提示词数据**：积累的高质量提示词和生成结果，用于优化模型和推荐。

### 核心诉求
- **创作者**：快速生成符合想象的视觉内容，操作门槛低，效果可控。
- **模型方**：提升模型能力，扩大用户规模和使用频率。
- **平台**：降低推理成本，通过会员和算力计费变现。
- **版权方**：确保生成内容不侵犯他人版权，防止深度伪造滥用。

### 关键矛盾
- **生成质量与算力成本**：高分辨率、高真实度的生成消耗大量 GPU，免费模式下难以持续。
- **创意控制与随机性**：用户希望精确控制输出，但生成模型有内在随机性。
- **版权归属与训练数据**：模型用公开图片训练，生成内容风格可能近似已有作品，引发版权争议。
- **创作民主化与深度伪造**：降低创作门槛的同时，也让虚假图片和视频的制作更容易。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：文生图工具模式
- **场景描述**：用户输入文字描述，AI 生成对应图片，支持风格选择和参数调节。
- **关键特性**：功能聚焦，用户上手快，付费点在于生成次数和高分辨率导出。

### 形态二：AI 视频生成模式
- **场景描述**：从文字描述或参考图生成短视频片段。
- **关键特性**：技术门槛更高，生成时间更长，适合创意预览和概念验证。

### 形态三：AI 设计工作流模式
- **场景描述**：AI 融入设计工具，实现智能抠图、风格迁移、画面扩展等辅助功能。
- **关键特性**：与专业设计工具结合，服务设计师和创意团队，企业付费意愿强。

### 形态四：社区+模型市场模式
- **场景描述**：用户分享作品和提示词，甚至可以上传微调模型进行交易。
- **关键特性**：用户粘性高，内容供给多样化，平台承担版权审核责任。
