# A2 - C2C闲置交易

## L1 领域核心模型

### 核心角色
- **卖家**：出售个人闲置物品的普通用户，非专业商家。
- **买家**：寻找二手商品的普通用户，追求性价比。
- **平台运营方**：提供信息展示、沟通工具、交易安全保障。

### 核心资源
- **闲置物品**：个人所有但不再使用的商品，每件通常是唯一的。
- **信用**：买卖双方的信用评分是交易安全的基石。
- **社区氛围**：用户之间的互动和信任感。

### 核心诉求
- **卖家**：快速变现闲置物品，卖个好价钱，操作简单，不被恶意买家欺诈。
- **买家**：以低价买到满意的商品，确认商品真实状态，不被卖家欺骗。
- **平台**：促成交易效率，降低纠纷率，通过增值服务（验货、担保）变现。

### 关键矛盾
- **低价与质量**：买家希望物美价廉，但低价往往伴随质量风险。
- **个人卖家与专业卖家**：平台通常限制商业行为，但难以区分个人闲置和囤货倒卖。
- **信息不对称**：卖家更了解商品瑕疵，买家只能通过图片和描述判断。
- **安全与便捷**：严格的身份验证提高安全，但增加用户使用门槛。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：纯信息撮合
- **场景描述**：平台仅提供展示和沟通渠道，交易由双方自行完成。
- **关键特性**：轻运营、难以抽佣、纠纷处理困难、体验不可控。

### 形态二：平台担保交易
- **场景描述**：平台托管交易款项，买家确认收货后打款给卖家。
- **关键特性**：安全度提升、可抽佣、需要处理纠纷。

### 形态三：C2B2C 寄卖/回收
- **场景描述**：平台或其合作伙伴从卖家处买入或寄卖，再卖给买家。
- **关键特性**：品控和定价权强、库存压力、需要质检和翻新能力。

### 形态四：社交化闲置交换
- **场景描述**：以物品交换代替买卖，或以积分/虚拟货币计价。
- **关键特性**：非现金交易、环保理念驱动、社区粘性强但变现困难。
