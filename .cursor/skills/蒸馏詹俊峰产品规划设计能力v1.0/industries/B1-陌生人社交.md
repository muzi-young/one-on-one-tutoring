# B1 - 陌生人社交

## L1 领域核心模型

### 核心角色
- **普通用户**：希望结识新朋友、寻找伴侣或扩大社交圈的个人。
- **平台运营方**：提供匹配算法、沟通工具和社区氛围管理。
- **内容审核方**：负责审核用户资料和发布内容，防止违规行为。

### 核心资源
- **用户资料**：照片、兴趣标签、个人介绍，是匹配的第一手信息。
- **匹配算法**：基于地理位置、兴趣、行为偏好等多维度的推荐引擎。
- **沟通渠道**：文字、语音、视频等即时通讯能力。

### 核心诉求
- **用户**：快速找到感兴趣的人，安全地沟通，不被骚扰或欺骗。
- **平台**：提升匹配成功率和用户活跃度，建立付费会员体系，控制虚假账号和不良行为。

### 关键矛盾
- **真实与隐私**：用户想了解对方真实信息，但又不想过度暴露自己。
- **开放与安全**：降低使用门槛吸引用户，但可能引入骚扰、诈骗等安全问题。
- **免费与付费**：基础功能免费吸引用户，高级功能付费变现，平衡点难以把握。
- **男性与女性用户体验**：男女用户比例失衡导致体验分化，女性被过度关注、男性得不到回应。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：滑动匹配模式
- **场景描述**：用户浏览他人资料卡片，左右滑动表示喜欢或跳过，互相喜欢则匹配成功。
- **关键特性**：操作简单、决策快速、颜值驱动明显、匹配后解锁聊天。

### 形态二：兴趣/灵魂匹配模式
- **场景描述**：通过问卷、测试或兴趣标签，系统推荐价值观或兴趣相投的人。
- **关键特性**：匹配质量更高、用户决策成本更高、适合寻找深度关系。

### 形态三：语音/视频社交模式
- **场景描述**：通过语音房间、视频连麦等方式实时互动，而非仅靠文字和照片。
- **关键特性**：真实感更强、社交效率更高、内容审核难度大。

### 形态四：游戏化社交模式
- **场景描述**：通过小游戏、互动玩法（如你画我猜、真心话大冒险）打破陌生人破冰。
- **关键特性**：降低尬聊概率、年轻用户偏好、游戏玩法需持续更新。
