# G5 - 宠物服务

## L1 领域核心模型

### 核心角色
- **宠物主**：饲养宠物的个人，需要宠物食品、用品、医疗和寄养服务。
- **服务商家**：宠物医院、美容店、寄养中心、宠物店等。
- **平台运营方**：提供商家聚合、社交社区和宠物档案管理。

### 核心资源
- **宠物档案**：宠物的品种、年龄、体重、疫苗记录等。
- **用户情感**：宠物主愿意为宠物支付较高费用，情感驱动消费。
- **社区内容**：宠物照片和视频分享形成的社区氛围。

### 核心诉求
- **宠物主**：宠物健康快乐，获得靠谱的服务和商品推荐。
- **服务商家**：获得稳定客流，展示专业能力，提升复购。
- **平台**：提升用户粘性和交易频次，通过电商、广告和服务佣金变现。

### 关键矛盾
- **情感消费与理性支出**：宠物主愿意为感情付溢价，但也可能在冲动消费后产生悔意。
- **服务质量与价格**：宠物行业缺乏统一标准，宠物主担心自己的毛孩子得不到妥善对待。
- **低频医疗与高频日常**：看病一年几次，主粮和零食月月都买，购买频次极不均。
- **社区繁荣与商业内容**：宠物社区天然适合照片分享，但过度商业化会破坏纯粹的氛围。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：宠物商城模式
- **场景描述**：在线购买主粮、零食、玩具、药品的一站式商城。
- **关键特性**：SKU 结构复杂，复购率高，适合会员订阅制。

### 形态二：服务预约模式
- **场景描述**：预约洗澡美容、绝育、寄养和体检等服务。
- **关键特性**：本地生活 O2O，运营重点是拓展足够数量的优质商家。

### 形态三：宠物社交模式
- **场景描述**：晒宠社区，同城遛狗组队，宠物相亲和领养平台。
- **关键特性**：用户活跃度极高，关系链沉淀价值大。

### 形态四：宠物医疗档案模式
- **场景描述**：数字化疫苗接种和体检档案，提醒驱虫和年度检查。
- **关键特性**：医疗健康工具属性，长期留存好，适合与保险等服务绑定。
