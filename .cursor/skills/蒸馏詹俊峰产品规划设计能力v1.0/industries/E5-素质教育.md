# E5 - 素质教育

## L1 领域核心模型

### 核心角色
- **儿童/青少年**：参与兴趣学习的主体。
- **家长**：买单者和选择者，最关注素质提升和考级结果。
- **兴趣教师/机构**：教授艺术、体育、编程、思维等非应试课程。
- **平台运营方**：匹配兴趣班、管理考勤和展示成果。

### 核心资源
- **教师/教练供给**：有特长且能教授儿童的老师是关键资源。
- **课程趣味性**：非学科辅导的素质教育核心吸引力在于好玩。
- **成果展示**：无论是作品墙、画展还是比赛，家长需要看到效果。

### 核心诉求
- **家长**：孩子开心，学到东西，最好有考级证书或比赛成绩。
- **教师/机构**：招满班，稳定开班，赚取课时费。
- **平台**：减少匹配摩擦，提高续费率。

### 关键矛盾
- **兴趣与考级**：过度追求考级成果会让课程变得枯燥，失去兴趣学习的本质。
- **师资标准化**：学科培训有统一大纲，素质教育教师水平参差不齐。
- **线下与线上化**：钢琴陪练和编程适合线上，但舞蹈和游泳几乎不可线上。
- **高单价与中低频**：相比学科培训，很多人觉得素质教育非刚需，不愿付出高额年费。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：线上约课平台
- **场景描述**：家长在线选课、选老师、选时间，上门教学或在机构完成。
- **关键特性**：灵活便捷，适合美术、钢琴陪练等可线上或上门教学的科目。

### 形态二：考级/比赛服务模式
- **场景描述**：平台聚焦考级报名、真题练习和比赛组织。
- **关键特性**：离家长不明确的考核诉求最近，变现路径清晰。

### 形态三：研学/营地教育模式
- **场景描述**：组织线下短期科学营、艺术营等，重实操和集体体验。
- **关键特性**：客单价极高，口碑传播强，但受季节和地域限制。

### 形态四：数字内容工作室模式
- **场景描述**：出售录播的舞蹈、绘画、手工视频教程包，配合送材料盒子。
- **关键特性**：边际成本低，容易快速起量，完成度依赖家长督促。
