# D7 - 心理健康

## L1 领域核心模型

### 核心角色
- **来访者/用户**：需要心理咨询、情绪支持或心理测评的个人。
- **心理咨询师/治疗师**：提供心理咨询服务的持证专业人员。
- **平台运营方**：匹配来访者和咨询师，提供咨询工具和支付支持。

### 核心资源
- **咨询师供给**：持证且有经验的心理咨询师，是服务的核心交付者。
- **咨询工具**：文字、语音、视频等多种咨询方式的可靠技术工具。
- **危机干预能力**：识别自杀自伤等高风险用户并进行干预的机制。

### 核心诉求
- **来访者**：找到匹配自己的咨询师，获得专业有效的帮助，负担得起费用。
- **咨询师**：获得稳定的来访来源，管理预约和记录，获得合理收入。
- **平台**：撮合匹配，保障服务质量，通过咨询佣金和增值服务变现。

### 关键矛盾
- **服务非标与质量保障**：咨询效果高度依赖咨询师个人，难以标准化。
- **供需失衡与匹配难度**：优秀的咨询师供不应求，匹配不合适导致脱落率高。
- **危机干预责任与平台能力**：高风险来访者的干预是法律和伦理要求，但平台能力有限。
- **付费意愿与经济负担**：心理咨询价格较高，不是所有人都愿意且有能力支付。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：咨询师平台模式
- **场景描述**：类似在线问诊，用户浏览咨询师资料，预约并进行在线咨询。
- **关键特性**：咨询师供给是核心，平台负责匹配和支付。

### 形态二：冥想/放松工具模式
- **场景描述**：以正念冥想、呼吸练习、助眠音频为核心功能。
- **关键特性**：不依赖咨询师，边际成本低，适合轻度情绪管理和压力缓解。

### 形态三：心理测评+课程模式
- **场景描述**：用户完成心理测评了解自己的状态，平台推荐对应课程。
- **关键特性**：获客成本低，测评的权威性和课程的转化率是关键。

### 形态四：EAP 企业服务模式
- **场景描述**：为企业提供员工心理援助计划（EAP），企业付费。
- **关键特性**：B2B 收入稳定，获客周期长，对企业预算依赖度高。
