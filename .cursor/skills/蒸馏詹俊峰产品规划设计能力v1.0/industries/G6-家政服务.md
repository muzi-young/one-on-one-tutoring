# G6 - 家政服务

## L1 领域核心模型

### 核心角色
- **消费者**：需要保洁、月嫂、育儿、维修等家庭服务。
- **服务人员/阿姨**：提供专业家庭服务的从业者。
- **家政公司/平台**：培训、匹配和管理服务人员与订单。

### 核心资源
- **优质服务供给**：经过背景核查和专业培训的服务人员。
- **服务标准体系**：清洁标准、礼仪要求、工具配备。
- **用户信任**：服务发生在私密家庭空间，信任是一切交易的前提。

### 核心诉求
- **消费者**：找到靠谱、安全、服务好的阿姨，定价明确。
- **服务人员**：稳定的单量和合理收入，人身安全有保障。
- **平台**：快速匹配，降低投诉率，控制跳单，通过佣金和增值服务变现。

### 关键矛盾
- **供给不稳定与需求波动**：年前扫除和节假日需求暴增，平时订单稀疏。
- **私约与平台留存**：阿姨和用户熟悉后容易私下交易，平台无法抽佣。
- **服务标准与主观评价**：干净不干净、态度好不好，极度主观，处理投诉复杂。
- **背调与人工成本**：深度背调让用户安心，但会增加审核成本和阿姨入职时间。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：平台匹配模式
- **场景描述**：用户下单，平台推荐空闲阿姨，服务完成后双方互评。
- **关键特性**：系统匹配效率高，但大促和节日时段明显供不应求。

### 形态二：自营员工模式
- **场景描述**：平台自聘阿姨，进行标准化培训和派单，控制服务质量。
- **关键特性**：服务质量更高更统一，但人力成本和管理复杂度极高。

### 形态三：垂直细分服务模式
- **场景描述**：只做月嫂、育儿、陪护或单一品类，强调高级技能证书和专业度。
- **关键特性**：客单价高，用户对服务人员的专业背景要求严格。

### 形态四：会员制/管家模式
- **场景描述**：家庭购买年费会员，长期固定保洁和家电清洗服务团队。
- **关键特性**：深度绑定用户，收入稳定，服务品质要求很高。
