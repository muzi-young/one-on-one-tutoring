# H2 - 长视频

## L1 领域核心模型

### 核心角色
- **观众**：观看剧集、电影、综艺等长视频内容的用户。
- **内容制作方**：影视公司、电视台、工作室等专业内容生产方。
- **平台运营方**：采购或自制版权内容，提供播放技术和会员服务。

### 核心资源
- **版权内容库**：独家或首播的优质剧集和电影，是付费转化的核心动力。
- **播放体验**：高清、流畅、多语言字幕、杜比音效等技术能力。
- **会员体系**：付费会员的数量和续费率决定平台收入稳定性。

### 核心诉求
- **观众**：看到想看的好内容，播放流畅，广告干扰少，值得付费。
- **内容制作方**：作品获得好的发行平台和合理回报。
- **平台**：控制内容采购成本，提升会员规模和ARPU，降低用户流失率。

### 关键矛盾
- **内容成本与盈利压力**：优质版权越来越贵，但会员涨价空间有限，导致长期亏损。
- **广告与会员体验**：广告收入需要曝光，但会员讨厌广告，平衡点难以把握。
- **独家版权与内容传播**：独家版权吸引会员，但限制了内容的社会影响力。
- **观众口味与内容生产**：爆款难以预测，大制作也可能扑街，投资风险高。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：SVOD 订阅制模式
- **场景描述**：用户按月或年付费，无广告观看所有内容。
- **关键特性**：收入稳定可预测，需要持续输出独家内容维持续费。

### 形态二：AVOD 广告制模式
- **场景描述**：内容免费观看，通过贴片、中插等广告实现收入。
- **关键特性**：用户基数可以很大，但单位用户价值低，追求高时长。

### 形态三：TVOD 单片付费模式
- **场景描述**：新上映的大片或演唱会，按次付费观看。
- **关键特性**：直接与内容热度挂钩，适合头部爆款内容。

### 形态四：IP 全链路开发模式
- **场景描述**：将剧集 IP 延伸到综艺、游戏、衍生品和主题乐园。
- **关键特性**：放大单 IP 价值，但对 IP 热度和运营能力要求极高。
