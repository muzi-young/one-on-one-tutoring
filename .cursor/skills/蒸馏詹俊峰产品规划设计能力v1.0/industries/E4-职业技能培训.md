# E4 - 职业技能培训

## L1 领域核心模型

### 核心角色
- **学员**：希望通过培训获得职业晋升、转行机会或技能提升的在职人士或学生。
- **讲师/导师**：具有行业实践经验的从业人员或专业讲师。
- **企业客户**：为员工采购培训服务的企业 HR 或管理者。
- **平台运营方**：提供课程体系、项目实战环境和就业对接服务。

### 核心资源
- **实战项目**：与真实工作场景接轨的案例和项目，是职培的核心差异化资源。
- **企业合作**：与招聘企业对接的渠道，关系学员毕业后的就业。
- **证书/认证**：学完课程获得的技能证明或学历认证。

### 核心诉求
- **学员**：学到能直接用于工作的技能，找到更好的工作或得到晋升。
- **企业**：提升员工能力，填补技能缺口，降低招聘成本。
- **平台**：提高付费转化和完课率，树立就业口碑，通过课程费和就业服务费变现。

### 关键矛盾
- **技能更新与课程开发**：行业技术变化快，课程容易过时，需要持续更新。
- **理论学习与实际操作**：职培强调动手中做，但线上学习难以提供真正的动手环境。
- **就业承诺与学员基础**：学员背景差异大，统一承诺就业可能导致纠纷。
- **与企业合作的深度**：深度合作能精准培养，但定制化成本高。浅层合作招工匹配度低。

## L2 动态形态参考

*请注意：以下为常见业务形态，AI 可根据需求自由组合、修改或颠覆。*

### 形态一：视频课程+项目实战模式
- **场景描述**：学员自主学视频，完成配套的线上或线下项目实战。
- **关键特性**：标准化程度高，适合编程、设计等可以线上实操的技能。

### 形态二：直播+助教训练营模式
- **场景描述**：固定时间开营，直播授课，学员在学习群中被督促和推动。
- **关键特性**：学员活跃度更高，氛围感更强，教学者投入的精力更多。

### 形态三：企业定向培训模式
- **场景描述**：平台根据合作企业的明确需求开设岗位定向班，结业直通面试或入职。
- **关键特性**：对就业的转化最直接，但对合作企业数量和质量要求极高。

### 形态四：校企合作/专业共建模式
- **场景描述**：与高校合作，平台承担实践教学部分，学分互认。
- **关键特性**：获客成本低，影响力大，决策周期长，受政策影响大。
