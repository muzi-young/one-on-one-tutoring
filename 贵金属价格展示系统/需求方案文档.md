# 五色土贵金属价格展示系统 - 需求方案文档

## 项目概述

### 客户信息

- **客户名称**: 五色土（2002年注册商标）
- **客户地址**: 上海豫园
- **业务类型**: 贵金属回收业务

### 项目目标

开发一个嵌入微信公众号的 H5 页面，实时展示贵金属价格，供客户查看参考。

## 数据源与计算逻辑

### 三、数据来源

#### 3.1 同花顺数据库

- **上海黄金交易所**
  - 黄金9999成交价（克）
  - 白银T+D成交价（千克）
  - 铂金9995成交价（克）
- **上海期货交易所**
  - 沪铜主连成交价（吨）
  - 沪锡主连成交价（吨）
- **伦敦市场**
  - 伦敦金价
  - 伦敦银价
  - 伦敦铜价（美元/吨）
  - 伦敦锡价（美元/吨）
  - 伦敦铂金价格（美元/盎司）
- **汇率**
  - 美元兑离岸人民币汇率

### 四、计算公式

#### 4.1 汇率换算

- 伦敦铜/锡价格：美元/吨 × 汇率 = 人民币/吨
- 伦敦铂金价格：美元/盎司 × 汇率 ÷ 31.1035 = 人民币/克

#### 4.2 基差计算

```
金价基差 = 黄金9999成交价 - 伦敦金人民币成交价 (元/克)
银价基差 = 白银T+D成交价 - 伦敦银人民币成交价 (元/克)
铂价基差 = 铂金9995卖1价 - 伦敦铂人民币成交价 (元/克)
铜价基差 = 沪铜次主连成交价 - 伦敦铜人民币成交价 (元/吨)
锡价基差 = 沪锡主连成交价 - 伦敦锡人民币成交价 (元/吨)
```

#### 4.3 比值计算

```
金银比 = 黄金9999成交价 / 白银T+D成交价
金铂比 = 黄金9999成交价 / 铂金9995成交价
金铜比 = 黄金9999成交价 / 沪铜次主连成交价
金锡比 = 黄金9999成交价 / 沪锡主连成交价
```

#### 4.4 二手价格计算（含参数）

```
回购价 = 数据源价格 ± 管理员设置的加减参数
销售价 = 数据源价格 × 管理员设置的乘数参数
```

## 注意事项

1. **数据实时性**: 价格每秒更新，需考虑服务器性能
2. **停盘处理**: 非交易时间数据保持最后价格不变
3. **参数即时生效**: 管理员修改参数后，用户端立即显示新价格
4. **移动端适配**: H5页面需要良好的移动端展示效果
5. **安全性**: 管理端需要登录验证，防止未授权修改
