# 五色土贵金属价格展示系统 - Demo

## 项目概述

为上海五色土贵金属回收公司开发的 H5 价格展示系统，嵌入微信公众号使用。

## Demo 文件

### 1. 用户端 H5 页面
- **文件**: `demo-user-h5.html`
- **功能**: 展示实时贵金属价格（金、银、铂、铜、锡）
- **特点**:
  - 无需登录，公开访问
  - 实时价格每秒更新
  - 一手大盘价 + 二手大盘价
  - 基差和比值自动计算
  - 移动端响应式设计
  - 深色主题，金色点缀

### 2. 管理端后台
- **文件**: `demo-admin.html`
- **功能**: 配置价格参数（回购/销售价格的加减值或乘数）
- **测试账号**: admin / 123456
- **特点**:
  - 简洁的后台界面
  - 13 个可配置参数
  - 实时预览效果
  - 修改即时生效

## 如何运行

直接在浏览器中打开 HTML 文件即可预览：

```bash
# macOS
open demo-user-h5.html
open demo-admin.html

# Windows
start demo-user-h5.html
start demo-admin.html
```

## 价格计算逻辑

### 一手价格
- 直接从同花顺数据库获取上海交易所和伦敦市场价格

### 二手价格
- **加减模式**: 基础价格 ± 管理员设置的加减值
- **乘数模式**: 基础价格 × 管理员设置的乘数

### 比值计算
- 金银比 = 黄金价格 / 白银价格
- 金铂比 = 黄金价格 / 铂金价格
- 金铜比 = 黄金价格 / 铜价格
- 金锡比 = 黄金价格 / 锡价格
